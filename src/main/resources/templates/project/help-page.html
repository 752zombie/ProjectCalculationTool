This page will guide you through the usage of the Project Management Tool

1) Fill out information marked by green arrows - then save

2) You can create subtasks for this task by going to 'View subtasks'

3) Now create your first subtask and save it.

4) Now we need someone to manage this subtask. Create your first employee so he can take care of this subtask

5) If you want you can also add a certain skill to this employee. This is not mandatory!

6) Now return to the subtask page

7) You can now add your employee to your subtask

8) Again you can add a needed skill to the subtask(This is not mandatory)

9) Now return to 'My first task'

10) The Project Management Tool has now calculated 'Total task hours' and estimated a 'Calculated end date' based on the given information about this task and it's subtasks
